# FMIPython-triviador-project
Соло или мултиплейър тривия игра създадена с python и pygame. Съдържа различни категории въпроси, които могат да са с множествен избор или числови, специални рундове носещи двойни точки, класически режим с 10 въпроса или безкраен(до първи сбъркан въпрос), жокери 50/50 или помощ от публиката, таблица с топ 10 резултата. 

## Режими на игра

* **Соло игра**
  * 10 въпроса или безкраен режим

* **Мултиплейър игра**:
  * Хостване и присъединяване в локална мрежа
  * 2 или повече играчи
  * Хостът избира категории въпроси и режим
  * В безкраен режим печели последният останал или този с повече точки при равенство
  * Наблюдателски режим за отпадналите играчи в безкраен режим

## Структура на проекта

* **triviador**: Главен пакет с игровата логика, потребителския интерфейс и данни за играта
* **tests**: pytest тестове

## Инструкции за стартиране

* **Изисквания**
  * python 3.10+
  * pip

* **Инсталация**
```bash
# Клонирай репото
git clone <repo-url>
cd triviador

# Създай виртуална среда
python -m venv .venv
source .venv/bin/activate   # Linux/macOS
.venv\Scripts\activate      # Windows

# Инсталирай в editable mode
pip install -e .
```

* **Стартиране**
```bash
python run_game.py
```
